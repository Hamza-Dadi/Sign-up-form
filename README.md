# Introduction JavaScript - Formulaire d'Inscription

Projet d'introduction à JavaScript avec un formulaire d'inscription fonctionnel.

## 📋 Fonctionnalités

- **Formulaire d'inscription** avec validation des champs
- **Validation du mot de passe** (minimum 8 caractères)
- **Vérification des champs obligatoires**
- **Stockage des données** en localStorage
- **Réinitialisation du formulaire** après inscription

## 🛠️ Technologies

- HTML
- CSS
- JavaScript

## 🚀 Utilisation

1. Ouvrir `index.html` dans le navigateur
2. Remplir le formulaire (nom, email, mot de passe)
3. Cliquer sur "S'inscrire"
4. Les données sont enregistrées localement
