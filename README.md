# Sign Up Form - Application d'Authentification

Projet d'introduction à JavaScript avec un système complet d'inscription et de connexion.

## 📋 Fonctionnalités

### Inscription (Signup)
- **Formulaire d'inscription** avec validation des champs
- **Validation du mot de passe** (minimum 8 caractères)
- **Vérification des champs obligatoires**
- **Stockage des données** en localStorage
- **Réinitialisation du formulaire** après inscription

### Connexion (Login)
- **Formulaire de connexion** avec email et mot de passe
- **Vérification des identifiants** en localStorage
- **Message de bienvenue** personnalisé
- **Lien vers l'inscription** pour les nouveaux utilisateurs
- **Validation des champs**

## 📄 Pages

- **Signup.html** - Formulaire d'inscription
- **login.html** - Formulaire de connexion

## 🛠️ Technologies

- HTML
- CSS
- JavaScript 
- LocalStorage API

## 🚀 Démarrage

1. Ouvrir `Signup.html` pour s'inscrire
2. Remplir le formulaire (nom, email, mot de passe minimum 8 caractères)
3. Cliquer sur "S'inscrire"
4. Ouvrir `login.html` pour se connecter
5. Entrer email et mot de passe pour valider la connexion
